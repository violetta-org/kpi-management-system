# 🚀 Power Platform React Code App - Task & KPI Management

This repository contains the **Code-First React + TypeScript Code App** (Task & KPI Management System) developed locally and connected to the Microsoft Power Platform `QLDA` environment.

Initially built using Vibe Portal (`vibe.powerapps.com`), the project has been successfully migrated to a native **Code App** to allow full layout control, component-level flexibility, and robust local testing.

---

## 📂 Project Structure

*   `code-app/`: The root directory of the React + TypeScript Code App.
    *   `src/`: React source code.
        *   `App.tsx`: Main dashboard and routing interface, supporting RBAC simulation.
        *   `index.css` & `App.css`: Pure white flat outline design system (with `#b6393a` accent, `50px` rounded card corners, and Fluent UI typography).
        *   `generated/`: Autogenerated TypeScript models and services representing connected Dataverse tables.
    *   `power.config.json`: Code App definition and registered Dataverse entities.
*   `src/`: Original Canvas Solution components (for reference/archival).

## 🛠️ Getting Started (Local Development)

### 1. Prerequisites
Ensure you have the following installed locally:
- Node.js (v18+)
- git
- Power Platform CLI (`pac`) - [Installation Guide](https://learn.microsoft.com/power-platform/developer/cli/introduction)

### 2. Installation
Navigate to the `code-app` directory and install the required dependencies:
```bash
cd code-app
npm install
```

### 3. Run Development Server
Start the local Vite development server:
```bash
npm run dev
```
By default, the server runs at [http://localhost:5173](http://localhost:5173).

## 🔄 Syncing Dataverse Tables (Data Sources)

All 33 Dataverse tables with the prefix `cr5db_` are synced locally. If you update the schema on Dataverse and need to regenerate the TypeScript services/models:

Run the Power Apps CLI to add/sync data sources:
```bash
node node_modules/@microsoft/power-apps-cli/dist/Bin.js add-data-source --org-url https://orgcaf78765.crm5.dynamics.com/
```

## 🎨 UI Design System & Guidelines

The application strictly implements the design tokens from `design-tokens-apps-powerapps-com.json`:
- **Canvas/Background:** Pure White (`#ffffff`)
- **Primary Accent / Highlight:** `#b6393a` (Crimson)
- **Secondary Accent:** `#b15058`
- **Text:** Pure Black (`#000000` / `#323130`)
- **Borders:** Thin black/dark-grey outlines (`#262626` / `#e5e5e5`)
- **Rounding:** Large `50px` radius for metric and layout cards, subtle `2px`/`4px` for buttons and inputs.
- **Typography:** `Inter` for headings, `sans-serif` for body text.
